Conteúdos do Curso EXTRA

Nestas aulas, você seguirá a jornada completa do desenvolvimento front-end. Começamos com o HTML Básico, onde você aprenderá a construir a fundação e a estrutura de qualquer página web. Em seguida, mergulhamos no CSS para dar vida ao seu projeto com estilos, layouts responsivos e animações. Para consolidar tudo, finalizamos com um Projeto Integrador, onde você aplicará 100% do que aprendeu para construir um site completo do zero.
